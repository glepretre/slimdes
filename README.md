# Slimdes

## Dependencies

* node
* yarn

## Installation

```sh
yarn install
```

## Usage

```sh
http-server
```

then open the page in your favorite browser :wink:

## Creating new slides

* Create a new branch
* Create a subdirectory in `slides/`
* Add a list item in `index.html`
  * ``<li><a href="slides/new/new.html">Your new presentation</a></li>``
